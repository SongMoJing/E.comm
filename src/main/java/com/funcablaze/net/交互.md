## 交互格式

|    头     | 对象         | 内容格式 |
|:--------:|------------|------|
|  `TEXT`  | 客户端 -> 客户端 | json |
|  `FILE`  | 客户端 -> 客户端 | json |
|  `CARD`  | 客户端 -> 客户端 | json |
|   `MD`   | 服务端 -> 客户端 | json |
|  `INFO`  | 服务端 -> 客户端 | json |
| `SERVER` | 服务端 <> 客户端 | json |

`Send:[头]:[对象(IP)]:"[内容]"`

内容：
> text
```json
{
  "return": "200",
  "info": {
    "text": "hello$[face.sa]",
    "icon": "$[]"
  }
}
```
