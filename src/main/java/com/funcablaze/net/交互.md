## 交互格式

|    头     | 对象         | 内容格式 |
|:--------:|------------|------|
|  `TEXT`  | 客户端 -> 客户端 | text |
|  `FILE`  | 客户端 -> 客户端 | json |
|  `CARD`  | 客户端 -> 客户端 | json |
|   `MD`   | 服务端 -> 客户端 | json |
|  `INFO`  | 服务端 -> 客户端 | json |
| `SERVER` | 服务端 <> 客户端 | json |

`Send:[头]:[对象(IP)]:"[内容]"`

### 内容：

> text

|  值   |  信息   |
|:----:|:-----:|
| img  |  图片   |
| link |  链接   |

```text
hello
$[img?D:/.LJX/Program/Java/E.comm/src/main/resources/icon.jpg]
$[link?https://github.com/SongMoJing/E.comm]
```

> file

|    值    |   单位    |
|:-------:|:-------:|
|  valid  |  秒级时间戳  |
|  size   |  尺寸：KB  |

```json
{
  "name": "main.java",
  "valid": 1731924553,
  "size": 1.78,
  "type": "Java Source",
  "url": "https://github.com/SongMoJing/E.comm"
}
```
